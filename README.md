# Phoenix Precision Drones Web V25 — Communication Template Pipeline

This build continues the production-grade portal and backend rollout with the communication system foundation.

- `ppd_department_email_templates`
- `ppd_communication_cases`
- `ppd_email_render_log`
- `ppd_twilio_number_config`
- `ppd_render_email_template()`
- `ppd_queue_email_from_template()`
- `ppd_classify_communication()`
- `ppd_process_incoming_communication()`

## Communication features

- Department-branded email templates for Sales, Operations, Safety, Finance, Maintenance, Pilot Network, and Support.
- Custom HTML header blocks generated by department.
- Compact topography-style footer based on the PPD brand direction: Aviation Intelligence. Precision Data.
- Incoming email/SMS/voice/web messages can be classified into department, intent, priority, and suggested action.
- Quote-related communications can flow into the quote intake pipeline.
- Outgoing emails are rendered and queued, but provider sending remains paused until the email account is funded.
- Twilio SMS and Voice configuration foundation is present but paused until Twilio funding/configuration is active.
